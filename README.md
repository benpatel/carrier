Responsive Image Demo

A simple HTML5 project demonstrating how to display responsive images using a separate CSS stylesheet.

Project Structure

**project**
 - index.html
    - README.md
    - css 
        - style.css
    - images
        - table.png
    
**Features**
 - HTML5 boilerplate
 - CSS reset
 - Responsive images
 - Mobile-friendly viewport
 - External stylesheet
 - Getting Started
 - Clone or download the project.
 - Open index.html in any modern web browser.

No installation or build tools are required.

Files
index.html
Contains the HTML structure and references the external stylesheet.

css/style.css
Contains the CSS reset and responsive image styling.

images/table.png
Sample image displayed on the page.
